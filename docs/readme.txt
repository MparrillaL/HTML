# Web Club Deportivo Atlético El Plantinar

Pequeña web para el club del barrio con secciones de fútbol 7, baloncesto y boxeo,
más una página aparte solo para **Boxeo El Plantinar**.

## Qué he ido aprendiendo

- Maquetar con HTML5 (header, nav, section, main, footer…).
- Jugar con **Grid, Flexbox y Bootstrap** (row, col, py, mt, ms-auto…).
- Poner imágenes de fondo y tarjetas con sombras y hover.
- Crear anclas internas (`href="#horario"`) para saltar por la página.
- Hacer páginas de “En mantenimiento” y un mini botón con JS que muestra un aviso.

Este Proyecto me ha servido para practicar estructura, estilos y un poco de Bootstrap.
